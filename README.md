# HitungHargaJual - Static Web App

Aplikasi dashboard dan kalkulator harga jual seller yang siap upload ke hosting static.

## Cara Upload ke Hosting Biasa

1. Extract file ZIP.
2. Upload isi folder `hitung-harga-jual-ready` ke `public_html`, `www`, atau root hosting Anda.
3. Buka domain Anda.

Tidak perlu build, Node.js, database, atau konfigurasi server.

## Fitur

- Dashboard seperti referensi screenshot
- Kalkulator harga jual marketplace
- Riwayat perhitungan tersimpan di browser menggunakan localStorage
- Halaman pembayaran/pricing
- Halaman afiliasi
- Halaman profil
- Halaman bantuan
- Responsive desktop dan mobile
- Optimasi ringan: HTML, CSS, JavaScript murni tanpa framework runtime

## Catatan Production

Versi ini sudah siap online sebagai MVP static. Untuk fitur login multi-user, database cloud, payment gateway, dan subscription sungguhan, perlu backend seperti Supabase, Firebase, Laravel, atau Node.js.
